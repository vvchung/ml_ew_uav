# 機器學習在電子戰：無人機對抗教學資源

此項聚焦「無人機對抗」的「機器學習在電子戰應用」可視化與故事化教材，適合無技術背景的國防教育者，並搭配互動活動提升學習興趣。

---

## 📌 專案內容
- **真實案例**：包括俄烏衝突、南海電子戰演習等情境。
- **動畫與模擬影片**：參考用影片資源。
- **圖片素材**：參考用無人機攔截示意圖等。
- **互動遊戲設計**：簡易圖形化防禦系統模擬遊戲構想。
- **類比解釋方法**：幫助以生活經驗理解電子戰概念。
